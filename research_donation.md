|------|------|
| 대상 | 국가, 지자체, 국방헌금, 사립학교(시설비/교육비/연구비), 사회복지공동모금회, 대한적십자사 등 |

| 이월 공제 | 10년간 이월 가능 |

### 1-4. 정치자금기부금 (조특법 §76)
| 구간 | 세액공제율 |
|------|-----------|
| 10만원 이하 | **100/110** (실질 전액 환급) |
| 10만원 초과 ~ 3,000만원 | **15%** |
| 3,000만원 초과 | **25%** |

- 중앙선관위 정치후원금센터(give.go.kr) 통해 기부 가능
- 신용카드, 카카오페이, PAYCO, 휴대폰결제 등 다양한 결제 수단 지원

### 1-5. 고향사랑기부금 (고향사랑기부금법)

#### 2025 귀속 기준
| 구간 | 세액공제율 |
|------|-----------|
| 10만원 이하 | **100%** (전액 환급) |
| 10만원 초과 ~ 500만원 (→ 2025년부터 2,000만원으로 상향) | **16.5%** |
| 특별재난지역 10만원 초과분 | **30%** |

#### 2026 귀속 기준 (개정, 십삼월 앱 반영 필수)
| 구간 | 세액공제율 |
|------|-----------|
| 10만원 이하 | **100%** (전액 환급) |
| 10만원 초과 ~ 20만원 | **44%** (신설) |
| 20만원 초과 ~ 2,000만원 (일반) | **16.5%** |
| 20만원 초과 ~ 2,000만원 (특별재난지역) | **33%** |

> **핵심 변화**: 2026년부터 20만원 기부 시 세액공제 14.4만원 + 답례품 6만원 = **20.4만원** 환급 효과. 기부금보다 돌려받는 금액이 더 크다.

#### 공통 조건
- 본인 **주민등록상 거주지 외** 지자체만 가능
- 연간 상한: 2,000만원 (2025년부터 상향)
- 답례품: 기부금의 **30%** 한도 (지역 특산물 중심)

### 2-4. API 연동 가능성
- 행안부가 고향사랑e음 API를 민간에 개방 중 (디지털서비스개방 > openservice.go.kr)
- 민간 참여기업 공모 → 시범사업 → 본사업 확대 진행
- 공감만세(위기브) 모델: **지자체와 직접 계약** → 수수료 약 **10%**
- 2025년 협력 지자체: 4개(2023) → 17개(2024) → **64개(2025)**

## 3. 십삼월 앱 연동 설계안

### 3-1. 고향사랑기부금 (최우선 연동 대상)

**왜 최우선인가?**
- 10만원 기부 → 전액 환급 + 3만원 답례품 = **실질 수익 3만원**
- 2026년부터 20만원 기부 → 전액 환급 효과 + 6만원 답례품 = **실질 수익 4천원+**
- 기존 연말정산 앱에서 기부 연결까지 제공하는 서비스 **거의 없음** → 차별점
- 답례품이라는 "보상"이 있어 유저 행동 유도가 쉬움

**연동 방식**
| 방식 | 설명 | 난이도 |
|------|------|--------|
| A. 딥링크 연결 | 고향사랑e음 또는 위기브 앱/웹으로 바로가기 | 낮음 |
| B. API 연동 | 고향사랑e음 공공API 활용, 앱 내 기부 프로세스 | 중간 |
| C. 민간 플랫폼 파트너십 | 위기브/웰로 등과 제휴, 수수료 공유 | 중간 |
| D. 직접 민간 플랫폼 등록 | 행안부 민간 참여기업 공모 직접 신청 | 높음 |

**MVP 추천**: A(딥링크) → B/C(파트너십)로 단계적 확장

**앱 내 UX 시나리오**
1. 연말정산 시뮬레이션 결과 화면에서 "추가 환급 기회" 표시
2. "고향사랑 기부하면 10만원 전액 돌려받고 + 답례품까지" 배너
3. 답례품 인기 순위 보여주기 (한우, 사과, 감귤 등)
4. 기부 바로가기 → 고향사랑e음/위기브 연결
5. 기부 완료 후 예상 환급액 재계산

**수익 모델**
- 기부 연결 수수료 (위기브 모델 참고: 약 10%)
- 지자체 광고/프로모션 비용
- 답례품 큐레이션 → 커머스 연계

### 3-2. 정치자금기부금

**연동 방식**
- 정치후원금센터(give.go.kr) 딥링크 연결
- 공개 API 없음 → 웹뷰 또는 외부 링크 방식만 가능
- "10만원 기부하면 전액 환급" 안내 + 바로가기

**앱 내 UX 시나리오**
1. 기부금 공제 시뮬레이션에서 "정치자금 10만원 = 전액 환급" 안내
2. 정치후원금센터 바로가기 버튼
3. 기부 후 금액 입력 → 환급 시뮬레이션 반영

### 3-3. 종교기부금

**연동 방식**
- 종교단체별 공식 기부 시스템 없음 (표준화 어려움)
- 유저가 **직접 금액 입력** → 한도 자동 계산
- 기부금 영수증 발급 안내 (홈택스 간소화 서비스 조회 연계)

**앱 내 UX 시나리오**
1. "종교단체 기부금이 있으신가요?" 입력 폼
2. 소득 대비 10% 한도 자동 계산 → 초과 시 경고
3. "기부금 영수증은 홈택스에서 자동 조회됩니다" 안내

### 3-4. 일반 기부금 (법정/지정)

**연동 방식**
- 사회복지공동모금회, 대한적십자사 등 주요 단체 딥링크
- 기부금 영수증 관리 가이드
- 홈택스 간소화 서비스 연결

**앱 내 UX 시나리오**
1. 주요 기부 단체 리스트 제공
2. 기부 금액 입력 → 소득 대비 한도 계산 (법정 100%, 비종교 30%)
3. 이월 공제 안내: "올해 한도 초과분은 10년간 이월 가능"

## 4. 경쟁사 분석 — 기부금 관련 기능

| 서비스 | 기부금 기능 | 기부 연결 | 비고 |
|--------|------------|-----------|------|
| **삼쩜삼** | 환급액 계산 | 공감만세와 캠페인 협업 | 세금 환급 특화 |
| **토스** | 연말정산 계산기 | 없음 | 금융 슈퍼앱 |
| **뱅크샐러드** | 고향사랑기부 가이드 콘텐츠 | 없음 (정보 제공만) | 자산관리 |
| **위기브** | 고향사랑기부 전문 | 직접 기부 처리 | 기부 전문 |
| **십삼월 (계획)** | 전체 기부금 시뮬레이션 + 기부 연결 | 고향사랑/정치자금 연결 | **유일한 원스톱** |

> **차별점**: 기존 서비스는 "계산" 또는 "기부" 중 하나만 수행. 십삼월은 **시뮬레이션 → 최적 기부 추천 → 기부 실행 → 환급액 재계산**을 한 흐름으로 제공.

## 5. 사업계획서 반영 포인트

### 핵심 메시지
> "기부도 하고, 환급도 받고, 답례품도 받는 — 원스톱 절세 기부 서비스"

### 차별화 요소
1. **기부 = 절세 수단**이라는 인식 전환 유도
2. 고향사랑기부금 연동 → 기존 연말정산 앱에 **없는 기능**
3. 답례품 큐레이션 → 커머스 요소로 앱 체류 시간 증가
4. 2026년 세법 개정 반영 → "20만원 기부 = 20.4만원 돌려받기" 마케팅 소재

### 지자체 협력 가능성
- 지자체 입장에서 고향사랑기부금 유치 = 세수 확보
- 십삼월 앱 내 지자체별 답례품 프로모션 → **B2G 수익 모델**
- 위기브 사례: 64개 지자체 협력, 수수료 약 10%

### 단계별 로드맵
| 단계 | 시기 | 내용 |
|------|------|------|
| Phase 1 | MVP | 기부금 시뮬레이션 + 딥링크 연결 (고향사랑e음, 정치후원금센터) |
| Phase 2 | 출시 후 3개월 | 위기브/웰로 등 민간 플랫폼 파트너십, 답례품 큐레이션 |
| Phase 3 | 출시 후 6개월 | 고향사랑e음 API 직접 연동, 앱 내 기부 프로세스 완성 |
| Phase 4 | 출시 후 12개월 | 행안부 민간 플랫폼 직접 등록, 지자체 B2G 영업 |
