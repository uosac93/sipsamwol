# 1. 소득공제

## 1-1. 근로소득공제 (자동 적용)

- **근거:** 소득세법 제47조
- **공제 방식:** 소득공제 (총급여에서 자동 차감)
- **월할/일할:** 해당 없음 (연간 총급여 기준 자동 계산)
- **중도퇴사 시:** 퇴사 시점까지의 총급여 기준으로 자동 계산

| 총급여 구간 | 공제율 |
|---|---|
| 500만원 이하 | 총급여의 70% |
| 500만원 초과 ~ 1,500만원 이하 | 350만원 + 초과분의 40% |
| 1,500만원 초과 ~ 4,500만원 이하 | 750만원 + 초과분의 15% |
| 4,500만원 초과 ~ 1억원 이하 | 1,200만원 + 초과분의 5% |
| 1억원 초과 | 1,475만원 + 초과분의 2% |

- **공제한도:** 2,000만원

## 1-2. 인적공제

### 1-2-1. 기본공제 (소득세법 제50조)

- **공제 방식:** 소득공제 (1인당 연 150만원)
- **월할/일할:** 적용 없음 (과세기간 종료일 12.31 기준 판단, 연액 공제)
- **중도퇴사 시:** 퇴사 시점에 전액 공제 (월할 안 함)

| 대상 | 나이요건 | 소득요건 |
|---|---|---|
| **본인** | 없음 | 없음 (무조건 적용) |
| **배우자** | 없음 | 연 소득금액 100만원 이하 (근로소득만 있는 경우 총급여 500만원 이하) |
| **직계존속** (부모/조부모/배우자의 부모) | 만 60세 이상 (1965.12.31 이전 출생) | 연 소득금액 100만원 이하 |
| **직계비속** (자녀/입양자) | 만 20세 이하 (2005.1.1 이후 출생) | 연 소득금액 100만원 이하 |
| **형제자매** (본인/배우자의) | 만 20세 이하 또는 만 60세 이상 | 연 소득금액 100만원 이하 + 주민등록상 동거 |
| **위탁아동** | 해당 과세기간 6개월 이상 양육 | 소득요건 동일 |
| **기초생활수급자** | 없음 | 소득요건 동일 |

**소득요건 상세:**
- 연간 소득금액 합계 100만원 이하
- 근로소득만 있으면 총급여 500만원 이하
- 양도소득, 퇴직소득 포함 판단
- 비과세소득, 분리과세소득은 제외
- **나이 기준일:** 2025년 12월 31일

**직계존속 동거 요건:**
- 주민등록상 동거가 원칙
- 별거 시에도 실질적으로 부양하고 있으면 가능 (취학, 질병요양, 근무상 형편, 사업상 형편)
- 직계존속이 독립된 생계를 유지하는 경우는 제외

### 1-2-2. 추가공제 (소득세법 제51조)

- **공제 방식:** 소득공제
- **월할/일할:** 적용 없음 (연액 공제)
- **중도퇴사 시:** 퇴사 시점에 전액 공제

| 구분 | 공제액 | 조건 |
|---|---|---|
| **경로우대** | 1인당 100만원 | 기본공제 대상자 중 만 70세 이상 (1955.12.31 이전 출생) |
| **장애인** | 1인당 200만원 | 기본공제 대상자 중 장애인 |
| **부녀자** | 50만원 | 종합소득금액 3,000만원 이하 + 여성 + (배우자 있거나 부양가족 있는 세대주) |
| **한부모** | 100만원 | 배우자 없이 기본공제 대상 직계비속/입양자 있는 경우 |

**중복 적용 규정:**
- 부녀자공제 + 한부모공제 중복 시 → **한부모공제만** 적용
- 경로우대 + 장애인은 중복 가능
- 장애인 + 부녀자/한부모 중복 가능

**장애인 범위 (소득세법 시행령 제107조):**
- 「장애인복지법」에 따른 장애인
- 「국가유공자 등 예우 및 지원에 관한 법률」에 따른 상이자
- 항시 치료를 요하는 중증환자 (의료기관 장애인증명서 발급 필요)

### 1-3-4. 주택임차차입금 원리금상환액 (전세대출)

- **근거:** 소득세법 제51조의3 제1항 제4호, 소득세법 시행령 제112조
- **공제 방식:** 소득공제
- **한도:** 연 400만원
- **공제율:** 원리금상환액의 40%
- **적용 조건:**
  - 과세기간 종료일(12.31) 기준 무주택 세대의 세대주
  - 세대주가 주택관련 공제를 받지 않는 경우 세대원도 가능
  - 국민주택규모(85㎡) 이하 주택 임차
  - 대출기관: 금융회사, 국민주택기금 등
  - 임대차계약서 상 입주일과 주민등록 전입일 중 빠른 날로부터 전후 3개월 이내 차입
- **월할/일할:** 적용 없음 (연간 상환액 기준)
- **중도퇴사 시:** 해당 연도 상환액 전액 기준으로 공제

### 1-3-5. 장기주택저당차입금 이자상환액 (주택담보대출)

- **근거:** 소득세법 제51조의3 제1항 제5호, 소득세법 시행령 제112조의2
- **공제 방식:** 소득공제
- **적용 조건:**
  - 과세기간 종료일 기준 1주택 이하 세대의 세대주 (세대주가 안 받으면 세대원 가능)
  - 취득 당시 기준시가 6억원 이하 (2019.1.1 이후 차입분은 5억원 이하)
  - 상환기간 15년 이상
  - 본인 명의 주택 + 본인 명의 차입금

| 구분 | 연간 공제한도 |
|---|---|
| 상환기간 15년 이상 + 고정금리 또는 비거치식 | 1,500만원 |
| 상환기간 15년 이상 + 고정금리 **AND** 비거치식 | 1,800만원 |
| 상환기간 10년 이상 15년 미만 + 고정금리 또는 비거치식 | 300만원 |
| 상환기간 10년 이상 15년 미만 + 고정금리 **AND** 비거치식 | 600만원 |

- **2024년 개정 (2024.1.1 이후 차입분):**

| 구분 | 연간 공제한도 |
|---|---|
| 상환기간 15년 이상 + 고정금리 또는 비거치식 | 1,500만원 |
| 상환기간 15년 이상 + 고정금리 **AND** 비거치식 | 2,000만원 |
| 상환기간 10년 이상 15년 미만 + 고정금리 또는 비거치식 | 300만원 |
| 상환기간 10년 이상 15년 미만 + 고정금리 **AND** 비거치식 | 600만원 |

- **기준시가 요건:** 2019.1.1 이후 차입분 5억원 이하
- **월할/일할:** 적용 없음 (연간 이자상환액 기준)
- **중도퇴사 시:** 해당 연도 이자상환액 전액 기준으로 공제

**전세대출 + 주담대 합산한도:**
- 전세대출 공제와 합산하여 적용 (각각 별도 한도)

## 1-4. 신용카드 등 사용금액 소득공제 (조세특례제한법 제126조의2)

- **공제 방식:** 소득공제
- **적용 조건:** 근로소득자 본인 및 기본공제 대상 배우자/직계존비속의 사용금액
- **최저사용금액:** 총급여의 25% 초과분부터 공제

### 결제수단별 공제율

| 결제수단 | 공제율 |
|---|---|
| 신용카드 | 15% |
| 체크카드/현금영수증 | 30% |
| 도서·공연·미술관·박물관 사용분 (총급여 7,000만원 이하만) | 30% |
| 전통시장 사용분 | 40% → **50% (2025년 한시 상향)** |
| 대중교통 사용분 | 40% → **80% (2024~2025년 한시 상향)** |
| 영화관람료 (총급여 7,000만원 이하만) | 30% → **40% (2025년 한시 상향)** |

### 공제한도

| 총급여 구간 | 기본한도 |
|---|---|
| 7,000만원 이하 | 300만원 |
| 7,000만원 초과 ~ 1.2억원 이하 | 250만원 |
| 1.2억원 초과 | 200만원 |

### 추가한도 (기본한도 초과 시 추가 공제)

| 항목 | 추가한도 |
|---|---|
| 전통시장 사용분 | 각각 100만원 (합산 300만원) |
| 대중교통 사용분 | 각각 100만원 |
| 도서·공연·미술관·영화 사용분 (총급여 7,000만원 이하) | 각각 100만원 |

- **2025년 총급여 대비 5% 초과 증가분:** 추가 10% 공제 (한도 100만원)
- **월할/일할:** 적용 없음 (연간 사용금액 기준이나, 중도퇴사 시 근무기간 사용분만)
- **중도퇴사 시:** 근무기간 중 사용한 금액만 공제 대상 (1.1 ~ 퇴사일)
- **제외 항목:** 사업관련비용, 자동차 구입비(중고차 10%만 인정), 해외사용분, 보험료, 교육비, 공과금, 상품권 구입비 등

## 1-7. 주택마련저축 소득공제 (조세특례제한법 제87조)

- **근거:** 조세특례제한법 제87조
- **공제 방식:** 소득공제
- **참고:** 이 조항은 2014.12.31 이전 가입한 근로자주택마련저축에 대한 것

- **중도퇴사 시:** 해당 과세기간 혼인신고 여부로 판단, 전액 공제

## 2-8. 중소기업 취업자 소득세 감면 (조세특례제한법 제30조)

- **근거:** 조세특례제한법 제30조
- **공제 방식:** 소득세 감면 (산출세액에서 감면)

### 감면율 및 기간

| 대상 | 감면율 | 감면기간 | 연간 감면한도 |
|---|---|---|---|
| **청년** (만 15~34세, 군복무기간 최대 6년 차감) | **90%** | 5년 | 200만원 |
| **60세 이상** | 70% | 3년 | 200만원 |
| **장애인** | 70% | 3년 | 200만원 |
| **경력단절여성** | 70% | 3년 | 200만원 |

- **적용 조건:**
  - 중소기업에 취업한 경우 (중소기업기본법상 중소기업)
  - 감면 대상 업종: 제조업, 건설업, 도매업, 음식점업 등 (일부 업종 제외: 전문직, 금융보험업 등)
  - 경력단절여성: 동일 기업 또는 동일 업종 재취업, 퇴직 후 2~15년 이내
  - 최초 적용 시 「중소기업 취업자 소득세 감면신청서」 제출 필요
- **감면 시작일:** 취업일이 속하는 달
- **월할/일할:** 감면기간 중 근무월수에 비례 (월할 적용)
- **중도퇴사 시:** 퇴사 시점까지의 근로소득에 대해 감면 적용

## 2-9. 외국납부세액공제 (소득세법 제57조)

- **중도퇴사 시:** 해당 과세기간 외국납부세액 전액 기준

# 3. 비과세 소득 (소득세법 제12조)

비과세 소득은 총급여에 포함되지 않으므로, 과세표준 자체가 줄어듦.

## 3-1. 식대 (소득세법 제12조 제3호 러목)

- **비과세 한도:** 월 20만원 (2023년 개정: 10만원→20만원)
- **적용 조건:**
  - 사내급식 또는 이와 유사한 방법으로 제공받지 않는 근로자
  - 식사를 제공받는 경우 비과세 불가 (현물 OR 현금 택1)
- **월할/일할:** 월별 적용 (근무한 달만)
- **중도퇴사 시:** 근무한 월수만큼 비과세

---

## 3-2. 자가운전보조금 (소득세법 제12조 제3호 자목, 시행령 제12조)

- **비과세 한도:** 월 20만원
- **적용 조건:**
  - 본인 명의 차량 (본인 또는 본인과 세대를 같이 하는 배우자 등의 명의)
  - 시내출장 등에 소요된 실제 여비를 지급받는 대신 자가운전보조금을 받는 경우
  - 차량유지비·유류대를 포함한 월정액
  - 업무 외 사적 사용분은 제외 (실무상 월정액 지급 시 20만원까지 비과세)
- **월할/일할:** 월별 적용
- **중도퇴사 시:** 근무한 월수만큼 비과세

---

## 3-3. 육아보육수당 (소득세법 제12조 제3호 타목)

- **적용 조건:**

  - 출산일 이후 2년 이내에 지급받는 경우
  - 2024년 1월 1일 이후 지급분부터 적용
- **월할/일할:** 해당 없음 (일시금)

# 4. 기부금 상세

## 4-1. 기부금 유형별 구분

### (1) 법정기부금 (소득세법 제34조 제2항, 시행령 제73조)

- **공제 방식:** 세액공제
- **한도:** 근로소득금액의 100%
- **세액공제율:** 1,000만원 이하 15%, 1,000만원 초과 30% (2024년 개정: 3,000만원 초과 시 35%)
- **대상:**
  - 국가 또는 지방자치단체에 무상으로 기증하는 금품
  - 국방헌금과 위문금품
  - 천재지변으로 인한 이재민 구호금품
  - 사립학교, 비영리교육재단 등에 시설비·교육비·장학금·연구비
  - 사회복지공동모금회, 대한적십자사
  - 독립기념관, 한국과학기술원 등
- **이월공제:** 10년

### (2) 지정기부금 — 비종교 (소득세법 제34조 제1항, 시행령 제80조)

- **공제 방식:** 세액공제
- **한도:** 근로소득금액의 30%
- **세액공제율:** 1,000만원 이하 15%, 1,000만원 초과 30%, 3,000만원 초과 35%
- **대상:**
  - 사회복지법인
  - 문화·예술·교육·학술 등 비영리법인
  - 「비영리민간단체 지원법」에 따른 비영리민간단체
  - 환경보전 관련 비영리법인
  - 의료법인
  - 기획재정부장관이 지정한 단체
- **이월공제:** 10년

### (3) 지정기부금 — 종교단체 (소득세법 제34조 제1항)

- **공제 방식:** 세액공제
- **한도:** 근로소득금액의 10%
- **세액공제율:** 1,000만원 이하 15%, 1,000만원 초과 30%
- **대상:** 「종교의 보급 기타 교화를 목적으로 설립된 비영리법인」에 기부하는 금품
- **이월공제:** 10년

### (4) 정치자금기부금 (조세특례제한법 제76조)

- **공제 방식:** 세액공제
- **세액공제율:**

| 기부금액 | 공제율 |
|---|---|
| 10만원 이하 | 100/110 (세액공제, 실질 전액 환급) |
| 10만원 초과 ~ 3,000만원 이하 | 15% |
| 3,000만원 초과 | 25% |

- **한도:** 근로소득금액의 100%
- **대상:** 정당, 정치인후원회, 선거관리위원회 등에 기부
- **이월공제:** 불가

### (5) 고향사랑기부금 (고향사랑기부금법, 조세특례제한법 제58조)

- **공제 방식:** 세액공제
- **세액공제율:**

| 기부금액 | 공제율 |
|---|---|
| 10만원 이하 | 100% (전액 세액공제) |
| 10만원 초과 ~ 500만원 이하 | 16.5% |

- **연간 한도:** 500만원 (2025년 개정: 500만원→2,000만원으로 확대)
- **대상:** 주민등록상 주소지 외의 지방자치단체에 기부
- **답례품:** 기부금의 30% 이내 답례품 제공 (10만원 기부 시 3만원 답례품 + 10만원 세액공제)
- **이월공제:** 불가

---

## 4-2. 기부금 세액공제율 정리

| 기부금 유형 | 1,000만원 이하 | 1,000만원 초과 ~ 3,000만원 | 3,000만원 초과 |
|---|---|---|---|
| 법정기부금 | 15% | 30% | 35% |
| 지정기부금(비종교) | 15% | 30% | 35% |
| 지정기부금(종교) | 15% | 30% | 30% |
| 정치자금(10만원 초과) | 15% | 15% | 25% |

- **2024년 개정:** 기부금 3,000만원 초과분 35% 공제율 신설 (법정·지정기부금)

## 4-3. 기부금 이월공제 (소득세법 제34조 제3항)

- 법정기부금, 지정기부금: **10년** 이월공제 가능
- 정치자금기부금, 고향사랑기부금: 이월공제 **불가**
- 이월순서: 먼저 발생한 기부금부터 순차적으로 공제

# 5. 중도퇴사/입사 관련

## 5-1. 중도퇴사자 연말정산 (소득세법 제137조)

- **퇴사 시 원천징수:**
  - 중도퇴사 시 퇴직하는 달의 급여를 지급할 때 연말정산 수행
  - 원천징수의무자(회사)가 퇴사 시점까지의 근로소득에 대해 정산
  - 퇴사 후 다른 근로소득이 없으면 이것으로 확정 (종합소득세 신고 불필요)
  - 단, 다른 소득이 있거나 추가 공제가 있으면 5월 종합소득세 신고 가능

## 5-2. 이직 시 합산 (소득세법 제138조)

- 전 직장 근로소득원천징수영수증을 현 직장에 제출
- 현 직장에서 합산하여 연말정산
- 미제출 시 각각 별도로 정산되어 과소 납부 → 5월 종합소득세 신고 시 합산

## 5-3. 근무기간 1년 미만 시 월할/일할 계산 기준

### 월할 계산 적용 항목

| 항목 | 월할 적용 여부 | 기준 |
|---|---|---|
| 근로소득공제 | X (자동계산) | 연간 총급여 기준 |
| 인적공제 (기본+추가) | **X (전액)** | 과세기간 종료일(또는 퇴사일) 기준, 연액 공제 |
| 국민연금·건강보험·고용보험 | X (실납부액) | 실제 납부한 금액 전액 |
| 주택청약저축 공제 | X (전액) | 연간 납입액 기준 |
| 주택임차차입금·장기주담대 | X (전액) | 연간 상환액 기준 |
| 신용카드 등 공제 | **O (근무기간)** | 근무기간 중 사용분만 (1.1~퇴사일 또는 입사일~12.31) |
| 보장성보험료 | X (전액) | 연간 납입액 기준 |
| 의료비 | **O (근무기간)** | 근무기간 중 지출분만 |
| 교육비 | **O (근무기간)** | 근무기간 중 지출분만 |
| 기부금 | X (전액) | 연간 기부금 기준 |
| 연금계좌 | X (전액) | 연간 납입액 기준 |
| 근로소득세액공제 | X (자동) | 산출세액 기준 자동 |
| 자녀세액공제 | **X (전액)** | 연액 공제 |
| 월세세액공제 | X (전액) | 연간 월세 지급액 기준 |
| 중소기업 취업자 감면 | **O (월할)** | 감면기간 중 근무월수 비례 |

### 핵심 원칙
- **소득공제:** 대부분 연액 공제 (월할 X)
- **세액공제 중 지출증빙형:** 근무기간 중 지출분만 (의료비, 교육비)
- **세액공제 중 정액형:** 연액 공제 (자녀세액공제, 결혼세액공제)
- **비과세:** 월별 한도 적용 (근무한 달만)

## 5-4. 일할 계산

- 일할 계산이 적용되는 항목은 **거의 없음**
- 비과세 소득(식대, 자가운전보조금 등): 월 단위 (일할 아님, 해당 월에 1일이라도 근무하면 월 전체 비과세)
- 중소기업 취업자 소득세 감면: 월할 (일할 아님)

# 6. 특수 케이스

## 6-1. 외국인 근로자 특례 (조세특례제한법 제18조의2)

- **근거:** 조세특례제한법 제18조의2
- **단일세율 선택:** 근로소득에 대해 **19% 단일세율** 적용 가능 (지방소득세 포함 20.9%)
- **적용 조건:**
  - 국내에서 근무를 시작한 날부터 **20년** 이내 (2023년 개정: 5년→20년)
  - 원천징수의무자에게 신청
  - 단일세율 선택 시 각종 공제·감면 적용 불가 (비과세, 소득공제, 세액공제 모두 포기)
- **선택 기준:** 총급여가 높아 실효세율이 19% 이상이면 단일세율 유리
- **매년 선택 가능:** 과세기간별로 일반과세와 단일세율 중 선택

---

## 6-2. 일용근로자 (소득세법 제129조 제1항 제3호)

- **근거:** 소득세법 제129조, 제137조의2
- **과세 방법:**
  - 일급에서 비과세(일 15만원 공제) 차감 → 6% 원천징수 → 55% 근로소득세액공제
  - 실효세율: 약 2.7% (= 6% × 45%)
  - 분리과세 (종합소득세 신고 불필요)
- **비과세:** 일 15만원 (근로소득공제)
- **연말정산:** 불필요 (일용근로소득은 원천징수로 납세의무 종결)
- **일용근로자 판단기준:**
  - 건설공사에 종사하는 자: 1년 미만 고용
  - 하역작업에 종사하는 자: 기간 불문
  - 그 외: 3개월 미만 고용 (3개월 이상 계속 고용 시 일반 근로자)

---

## 6-3. 2개 이상 근무지 (주·종 사업장)

- **근거:** 소득세법 제140조
- **주(主) 사업장:** 공제 적용 및 연말정산 수행
- **종(從) 사업장:** 기본세율로 원천징수만 수행, 공제 미적용
- **연말정산:**
  - 종 사업장 근로소득원천징수영수증을 주 사업장에 제출
  - 주 사업장에서 합산하여 연말정산
  - 미합산 시 5월 종합소득세 신고에서 합산
- **공제 적용:**
  - 인적공제: 주 사업장에서만 적용 (중복공제 불가)
  - 신용카드 등: 합산 총급여 기준으로 계산
  - 각종 세액공제: 주 사업장에서 합산 적용
