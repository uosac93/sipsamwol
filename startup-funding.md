## 1. 예비창업패키지 (최우선 추천)

### 개요
- **주관**: 중소벤처기업부 / 창업진흥원
- **목적**: 혁신적 기술 창업 아이디어를 보유한 예비창업자 사업화 지원
- **지원금액**: 최대 1억원 (현실적 평균 선정 금액 약 4,500만~6,000만원, 1억 만점은 극히 드묾)
- **지원내용**: 사업화 자금 + 창업교육 + 멘토링

### 2026년 예상 공고 시기
- **1차 공고**: 2026년 1월 중순 ~ 2월 초 (접수: 2~3월)
- **2차 공고**: 2026년 5~6월 (일부 연도 시행)
- **현재 시점(3/31) 기준**: 1차 접수는 거의 확실히 마감됨. 즉시 K-Startup에서 2차 공고 여부 확인 필요
- 2025년 기준 1차 공고: 1월 22일, 접수 마감: 2월 28일

### 대표님 자격 분석
| 항목 | 조건 | 대표님 | 판정 |
|------|------|--------|------|
| 사업자등록 | 미등록 예비창업자 | 사업자등록 없음 | 적합 |
| 창업이력 | 대표이사 경험 없음 | 첫 창업 | 적합 |
| 연령 | 제한 없음 (청년 가점) | 만 32세 | 적합 + 가점 |
| 기술분야 | 혁신기술 기반 | AI/핀테크 | 적합 |
| 재직상태 | 신청 시 재직 가능, 선정 후 퇴사 및 사업자등록 필요 | 재직 중 | 가능 |

### 청년 가점
- 만 39세 이하 청년 가점 존재 (서면평가 시 약 2~5점 가점, 100점 만점 기준)
- 대표님은 만 32세로 청년 가점 확보
- 2025년부터 청년 특화 트랙 별도 운영 중 (청년 전용 배정 물량 있음)

### 세무사 자격의 유리한 점
- **전문성 입증**: 세무/회계 분야 국가공인 자격 = 사업 아이템의 도메인 전문성 직접 증명
- **실현 가능성**: 평가위원 입장에서 "세무사가 만든 세무테크 앱"은 실현 가능성 최상급
- **차별화**: 단순 개발자 창업 vs. 도메인 전문가 창업 → 사업계획서에서 강력한 어필 포인트
- **재무계획 신뢰도**: 세무사가 작성한 재무계획은 그 자체로 신뢰도 확보

### 프로토타입 완성의 가점
- **서면평가 핵심 가점 요소**: "사업 준비성" 항목에서 높은 점수
- 아이디어만 있는 지원자 vs. 프로토타입 + 웹앱 배포 완료 → 명확한 우위
- 시연 가능한 데모가 있으면 발표평가에서도 큰 강점
- 2025년 기준, 시제품/MVP 보유 시 상위 평가 경향 뚜렷

### 경쟁률 (최근 3년)
| 연도 | 신청 | 선정 | 경쟁률 |
|------|------|------|--------|
| 2023년 | 약 22,000명 | 약 2,400명 | 약 9.2:1 |
| 2024년 | 약 25,000명 | 약 2,500명 | 약 10:1 |
| 2025년 | 약 27,000명 | 약 2,600명 | 약 10.4:1 |

- 경쟁률 상승 추세이며, 난이도는 "중상~상" 수준. 단, 대표님 프로필(세무사+프로토타입+청년+핀테크)은 상위 경쟁력
- **주의**: 위 경쟁률 수치는 공개 정보 기반 추정치. 2026년 실제 수치는 K-Startup 공고 후 확인 필요

### 선정 후 의무사항
1. **사업자등록**: 선정 통보 후 약 2개월 이내 사업자등록 필수
2. **퇴사**: 사업자등록 시점까지 퇴사 완료 필요 (대표이사 전념 의무)
3. **전담기관 지정**: 창업진흥원 또는 지역 주관기관 소속
4. **사업비 집행**: 정부지원금은 반드시 사업 목적에 맞게 집행 (인건비, 외주비, 마케팅비 등)
5. **기간**: 약 10개월 (선정 후 ~ 당해 12월)
6. **성과보고**: 중간보고 + 최종보고 필수
7. **사후관리**: 종료 후 3년간 매출/고용 등 성과 보고

### 재직 중 신청 및 퇴사 시점
- **신청 시**: 재직 중 신청 가능 (사업자등록 안 된 상태면 OK)
- **서면평가 → 발표평가**: 재직 상태 무관
- **최종 선정 후**: 협약 체결 전까지 사업자등록 + 퇴사 완료
- **현실적 퇴사 시점**: 최종 선정 통보(보통 5~6월) 후 퇴사해도 충분
- **핵심**: 불합격 시 퇴사할 필요 없으므로, 선정 확정 후 퇴사가 안전한 전략

## 2. 초기창업패키지

### 개요
- **주관**: 중소벤처기업부 / 창업진흥원
- **목적**: 창업 3년 이내 초기 스타트업 사업화 지원
- **지원금액**: 최대 1억원 (평균 약 6,000만~8,000만원)
- **조건**: 사업자등록 후 3년 이내

### 대표님 해당 여부
- 현재 사업자등록 없음 → **지금 바로 신청 불가**
- 예비창업패키지 선정 → 사업자등록 → 이후 초기창업패키지 신청 가능
- 또는 사업자등록 먼저 한 후 초기창업패키지 직접 신청 가능

### 예비 → 초기 순서 전략
1. 2026년 예비창업패키지 선정 (최대 1억)
2. 사업자등록 (2026년 중반)
3. 2027년 초기창업패키지 신청 (최대 1억 추가)
4. 총 최대 2억원 확보 가능

### 주의사항
- 예비창업패키지와 초기창업패키지는 **동시 수혜 불가** (중복 제한)
- 순차적으로는 가능 (예비 종료 후 초기 신청)
- 예비창업패키지 수혜자도 초기창업패키지 지원 가능 (단, 평가 시 기수혜 이력 감안)

### 2026년 예상 공고
- **공고 시기**: 2026년 2~3월
- **접수**: 3~4월
- **선정**: 5~6월
- 대표님 기준: 사업자등록 후 신청해야 하므로 2027년 목표가 현실적

## 3. 청년창업사관학교

### 개요
- **주관**: 중소벤처기업부 / 창업진흥원
- **목적**: 만 39세 이하 청년 예비창업자 집중 육성
- **지원금액**: 최대 1억원 (사업화 자금) + 사무공간 + 코칭
- **기간**: 약 10개월 (입교~수료)
- **특징**: 입교 후 전용 사무공간 제공, 1:1 담당 코치 배정

### 대표님 자격 분석
| 항목 | 조건 | 대표님 | 판정 |
|------|------|--------|------|
| 연령 | 만 39세 이하 | 만 32세 | 적합 |
| 사업자등록 | 미등록 예비창업자 또는 등록 1년 미만 | 미등록 | 적합 |
| 기술 | 혁신기술 기반 | AI/핀테크 | 적합 |

### 입소 조건
- 선정 후 해당 지역 창업사관학교에 **상주 필수** (주중 출석)
- 경기도 캠퍼스: 연도별로 안산/수원/성남 등 변경 가능 (2026년 공고에서 확인 필수)
- 전국 캠퍼스 중 선택 가능한 경우도 있으므로 "안산 고정"으로 단정 불가
- **하남시 거주 → 배정 캠퍼스에 따라 통근 또는 인근 거주 필요**

### 재직자 신청 가능 여부
- 신청 시 재직 가능
- **입교 시점까지 퇴사 필수** (전일제 참여 의무)
- 선정 → 입교 전 퇴사 전략 가능

### 2026년 예상 공고
- **공고**: 2026년 1~2월
- **접수**: 2~3월
- **현재 시점(3/31)**: 2026년 1차 접수 마감 가능성 높음, 추가 모집 확인 필요

### 경쟁률
- 약 8:1 ~ 12:1 (지역/연도별 차이)
- 청년 전용이라 예비창업패키지보다 경쟁률 낮은 편

### 예비창업패키지 vs. 청년창업사관학교
| 비교항목 | 예비창업패키지 | 청년창업사관학교 |
|----------|---------------|-----------------|
| 금액 | 최대 1억 | 최대 1억 |
| 공간 | 미제공 | 전용공간 제공 |
| 전일제 | 아님 | 주중 상주 필수 |
| 코칭 | 멘토링 | 1:1 전담 코치 |
| 중복신청 | 동시 불가 | 동시 불가 |

**추천**: 대표님은 하남 거주 + 재직 중이므로 예비창업패키지가 더 현실적 (상주 불필요)

- **하남시 소상공인 창업지원**: 최대 2,000만원 (업종 제한 확인 필요)

- **확인처**: 하남시청 일자리경제과 (031-790-6354)
- **특이사항**: 지자체 사업은 중기부 사업과 **중복 수혜 가능한 경우 많음** (반드시 공고 확인)

- **금액**: 300만~1,000만원 (연도별 상이)
- **특징**: 사업비가 아닌 '생활안정자금' 성격인 경우도 있음
- **공고 시기**: 상반기 (3~5월)
- **확인처**: 경기도 일자리재단 (https://www.gwork.kr/)

### 4-4. 경기도 판교 스타트업캠퍼스

- 핀테크 분야 스타트업 입주 프로그램

- 하남에서 판교 접근성 양호 (30분 거리)

## 5. 핀테크 특화 지원

### 5-1. 금융위원회 혁신금융서비스 지정

- **개요**: 기존 금융규제 적용 없이 혁신적 금융서비스 테스트 가능
- **대상**: 금융 관련 혁신기술 보유 기업
- **십삼월 해당 가능성**: 연말정산 환급 관련 서비스 → 금융데이터/마이데이터와 연계 시 해당 가능
- **지정 혜택**: 규제 면제/유예 (최대 4년) + 신뢰도 확보 + 투자 유치 유리
- **신청 시기**: 수시 접수
- **신청처**: 금융위원회 혁신금융서비스 심사위원회
- **주의**: 사업자등록 + 법인 설립 후 신청 가능
- **추천 시기**: 서비스 정식 출시 후 (2027년 이후 검토)

### 5-2. 금융규제 샌드박스

- 혁신금융서비스의 한 유형
- 새로운 금융서비스를 제한된 조건에서 시범 운영
- **십삼월 적용 가능 시나리오**:
  - AI 기반 세무 자문 서비스 → 세무사법과의 규제 이슈 해결
  - 환급금 선지급 등 금융서비스 연계 시

### 5-3. 한국핀테크산업협회 지원

- **핀테크 스타트업 지원 프로그램**: 연 1~2회 모집
- **내용**: 사업화 자금(1,000만~3,000만원) + 핀테크 데모데이 + 투자사 연계
- **장점**: 핀테크 특화 네트워크, VC/금융사 매칭
- **확인처**: https://korfin.co.kr/
- **공고 시기**: 보통 상반기 (3~5월)

### 5-4. 서울핀테크랩

- 마포구 소재, 핀테크 스타트업 전용 보육 공간
- 입주 + 멘토링 + 핀테크 규제 상담 + 금융사 협업
- 대표님 거주지 하남이지만 서울 접근 가능하면 검토 가치
- **공고**: 연 1~2회 (상반기)
- **확인처**: 서울핀테크랩 홈페이지

## 6. TIPS (Tech Incubator Program for Startup)

### 개요
- **주관**: 중소벤처기업부
- **금액**: 최대 5억원 (R&D 자금) + 창업사업화 1억원 + 해외마케팅 1억원
- **카테고리별 최대 합산**: 약 7억원이나, 실제 평균 수혜 규모는 총 3~5억원 수준
- **핵심 조건**: TIPS 운영사(민간 액셀러레이터/VC)의 투자 확약 필수

### 프로세스
1. TIPS 운영사에 사업 아이템 제안
2. 운영사가 투자 결정 (최소 1,000만원 이상 민간투자)
3. 운영사가 TIPS 추천
4. 중기부 평가 후 선정

## 10. 주의사항

### 재직 중 창업지원금 신청 시 퇴사 시점
- **신청 단계**: 퇴사 불필요 (재직 중 신청 가능)
- **선정 후**: 협약 체결 전까지 퇴사 + 사업자등록 필수
- **안전 전략**: 선정 확정 통보 받은 후 퇴사 (불합격 리스크 제거)
- **현 직장 주의**: 재직 중 타 사업 준비 시 근로계약서상 겸업금지 조항 확인
- **4대보험**: 퇴사 후 지역가입자 전환 또는 창업기업 대표로 가입
- **K-Startup 실명 노출 리스크**: 사업계획서를 실명으로 제출하므로, 세무법인 측에서 알게 될 가능성은 낮지만 0은 아님. 겸업금지 조항이 있다면 사전에 리스크 인지 필요

### 중복 수혜 제한
- **동일 부처 동일 시기 중복 불가**: 예비창업패키지 + 초기창업패키지 동시 수혜 불가
- **예비창업패키지 + 청년창업사관학교**: 동시 불가 (택 1)
- **중기부 사업 + 지자체 사업**: 공고별 상이 (일부 가능, 반드시 공고문 확인)
- **중기부 사업 + 핀테크협회**: 일반적으로 가능 (민간 지원 성격)
- **원칙**: 동일한 사업 내용으로 중복 지원 시 환수 위험 → 각 사업의 사업비 항목을 구분하여 집행

### 사업자등록 시점
- **너무 빠른 등록의 위험**: 사업자등록하면 예비창업자 자격 상실 → 예비창업패키지 신청 불가
- **최적 시점**: 예비창업패키지 선정 확정 후 등록
- **등록 형태**:
  - 개인사업자: 설립 간편, 비용 없음
  - 법인(주식회사): 투자 유치에 유리, 설립비 약 30~50만원
  - **추천**: 법인 설립 (향후 투자 유치, TIPS 등 고려)
- **업종코드**: 소프트웨어 개발업 (J62010), 정보서비스업 등

### 세금 이슈 — 지원금은 과세 대상인가?
- **정부 창업지원금**: 원칙적으로 **과세 대상** (사업소득)
- **그러나**: 중기부 창업지원금은 "출연금" 성격 → 사업비 집행 후 잔액 반납 구조
- **실질 과세**: 지원금으로 집행한 비용은 경비 처리 → 실질 세부담 낮음
- **인건비**: 대표자 인건비로 지급받은 금액은 근로소득 또는 사업소득으로 과세
- **부가가치세**: 지원금 자체에 부가세 없음. 지원금으로 구매한 재화/용역의 매입세액은 공제 가능
- **세무사이시니**: 이 부분은 대표님이 가장 잘 아실 것. 다만 집행 시 증빙 철저히 보관 필수

### 기타 주의
- **사업계획서 유출 주의**: K-Startup 등에 업로드한 사업계획서는 평가위원에게 공개됨. 핵심 기술 노하우는 별도 보호
- **특허/지식재산**: 가능하면 사업자등록 전 특허 출원 (발명자: 박양훈 개인) → 이후 법인에 양도
- **세무사 자격과 앱 서비스**: 세무사법상 세무 상담/신고 대리는 세무사만 가능 → AI 자동 세무 자문이 세무사법 저촉 여부 사전 검토 필요
- **근로소득자 부업/겸업**: 현 직장 취업규칙상 겸업금지 조항 확인. 사업자등록 = 겸업으로 볼 수 있음

## 11. IT 인프라 비용 사업비 집행 가이드

### 예비창업패키지 사업비 항목별 비율 제한

| 항목 | 비율 제한 | 포함 가능 내용 |
|------|----------|---------------|
| 인건비 | 총 사업비의 50% 이내 | 대표자 인건비(월 최대 약 250만원), 고용 직원 인건비 |
| 재료비 | 제한 없음 | 원자재, 소모품, 시제품 제작 |
| 외주용역비 | 총 사업비의 50% 이내 | 개발 외주, 디자인 외주 등 |
| 지식재산권비 | 300만원 이내 | 특허/상표 출원 비용 |
| 마케팅비 | 총 사업비의 30% 이내 (통상) | 광고, 홍보, 마케팅 |
| 기자재 구입비 | 총 사업비의 40% 이내 | 장비, SW 라이선스 |
| 기타 운영비 | 보통 10~20% 이내 | 서버비, 클라우드, SaaS 구독료 |

### API 호출료/클라우드 비용을 사업계획서에 넣는 방법

**원칙: 브랜드명이 아닌 기능 중심으로 기술**

| 실제 비용 | 사업계획서 기재명 | 분류 항목 |
|-----------|-----------------|----------|
| Claude API 호출료 | "LLM 기반 세무 상담 엔진 API 이용료" 또는 "AI 자연어처리 엔진 API 서비스 이용료" | 기타 운영비 또는 재료비 |
| Claude 구독료(Pro 등) | "AI 개발도구 구독 서비스 이용료" | 기타 운영비 |
| AWS/GCP 서버비 | "클라우드 서버 호스팅 비용" | 기타 운영비 |
| Vercel/Firebase 등 | "클라우드 인프라 서비스 이용료" | 기타 운영비 |
| 도메인/SSL | "웹서비스 운영 비용" | 기타 운영비 |
| GitHub Copilot 등 | "AI 기반 개발 생산성 도구 이용료" | 기타 운영비 |

### 작성 시 주의사항

1. **"클로드 구독료"라고 절대 쓰지 말 것** → "AI 자연어처리 엔진 API 서비스 이용료"
2. **월별 예상 사용량과 단가를 반드시 명시**: 예) "AI API 호출 월 10만 건 x 건당 5원 = 월 50만원 x 10개월 = 500만원"
3. **기타 운영비에 IT 인프라를 과도하게 몰아넣으면 감점** → 전체 사업비의 15~20% 이내 유지 권장
4. **증빙 철저 보관**: 카드결제 내역, 세금계산서, 이용 내역 캡처 등 (해외 SaaS는 카드결제 내역 + 이용 화면 캡처)
5. **달러 결제 주의**: 환율 변동으로 인한 차액은 정산 시 이슈가 될 수 있음. 원화 결제 가능 서비스 우선 활용

### 예시: 사업비 6,000만원 기준 IT 인프라 배분 (참고)

| 항목 | 월 비용 | 10개월 합계 |
|------|---------|------------|
| AI API 이용료 (Claude 등) | 50만원 | 500만원 |
| 클라우드 서버 (AWS/GCP) | 30만원 | 300만원 |
| 기타 SaaS 도구 | 20만원 | 200만원 |
| **IT 인프라 소계** | **100만원** | **1,000만원 (전체의 약 17%)** |

## 12. 대표님 프로필 강점 요약 (사업계획서 어필 포인트)

1. **도메인 전문성**: 세무사 자격 + 세무법인 실무 2년 5개월 → "연말정산을 가장 잘 아는 창업자"
2. **실행력 입증**: 프로토타입 완성 + 웹앱 배포 → "말만 하는 게 아니라 만들어낸 사람"
3. **시장 이해도**: 실제 세무 현장에서 고객 니즈를 직접 체감 → 시장 검증 근거
4. **청년 + 첫 창업**: 가점 요소 + 정책적 지원 의지 높은 타겟
5. **핀테크/AI 트렌드**: 정부 핵심 육성 분야 → 선정 우선순위 높음
6. **사회적 가치**: 근로소득자의 세금 환급 극대화 → 국민 경제적 이익 → 정책적 의미
