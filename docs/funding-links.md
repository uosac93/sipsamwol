## 추가 확인 권장 사항

대부분의 주요 1차 공고가 마감된 상태이므로, 다음을 확인하시기 바랍니다:

1. **추가모집/수시모집 공고** - K-Startup 포털에서 수시 확인
2. **하반기 공고** - 초기창업패키지, 창업도약패키지 등 별도 일정
3. **지자체별 개별 공고** - 서울/경기/부산 등 자체 창업지원사업
4. **TIPS 운영사 직접 컨택** - 149개 운영사에 직접 사업계획서 제출 가능 (상시)
