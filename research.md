## 1. 소득공제 전체 항목

### 1-1. 근로소득공제 (자동 적용)

> 근거: 소득세법 제47조

총급여에서 자동으로 차감. 근로자가 별도로 신청할 필요 없음.

| 총급여 구간 | 공제율 |
|---|---|
| 500만원 이하 | 총급여의 70% |
| 500만원 초과 ~ 1,500만원 이하 | 350만원 + 500만원 초과분의 40% |
| 1,500만원 초과 ~ 4,500만원 이하 | 750만원 + 1,500만원 초과분의 15% |
| 4,500만원 초과 ~ 1억원 이하 | 1,200만원 + 4,500만원 초과분의 5% |
| 1억원 초과 | 1,475만원 + 1억원 초과분의 2% |

- 공제한도: 2,000만원

### 1-2. 인적공제

#### 1-2-1. 기본공제 (1인당 150만원)

> 근거: 소득세법 제50조

| 대상 | 요건 |
|---|---|
| 본인 | 무조건 적용 |
| 배우자 | 연간 소득금액 100만원 이하 (근로소득만 있는 경우 총급여 500만원 이하) |
| 직계존속 (부모/조부모) | 만 60세 이상 + 소득요건 동일 + 주민등록상 동거 또는 별거 시 실질 부양 입증 |
| 직계비속 (자녀/입양자) | 만 20세 이하 + 소득요건 동일 |
| 형제자매 | 만 20세 이하 또는 만 60세 이상 + 소득요건 동일 + 주민등록상 동거 |
| 위탁아동 | 해당 과세기간 6개월 이상 직접 양육 |
| 수급자 | 「국민기초생활보장법」에 따른 수급자 |

**소득요건 상세:**
- 연간 소득금액 합계 100만원 이하
- 근로소득만 있으면: 총급여 500만원 이하
- 양도소득, 퇴직소득 포함하여 판단
- 비과세소득, 분리과세소득은 제외

**나이 기준일:** 해당 과세기간 12월 31일 (2025년 12월 31일 기준)

#### 1-2-2. 추가공제

> 근거: 소득세법 제51조

| 구분 | 공제액 | 조건 |
|---|---|---|
| 경로우대 | 1인당 100만원 | 기본공제 대상자 중 만 70세 이상 (1955.12.31 이전 출생) |
| 장애인 | 1인당 200만원 | 기본공제 대상자 중 장애인 (소득세법 시행령 제107조) |
| 부녀자 | 50만원 | 종합소득금액 3,000만원 이하인 여성 근로자로서 배우자가 있거나 부양가족이 있는 세대주 |
| 한부모 | 100만원 | 배우자 없이 기본공제 대상 직계비속 또는 입양자가 있는 경우 |

- 부녀자 + 한부모 중복 시: **한부모공제만** 적용 (중복 불가)
- 장애인 범위: 「장애인복지법」 장애인 + 「국가유공자법」 상이자 + 항시 치료를 요하는 중증환자 (의료기관 장애인증명서 발급)

### 1-6. 주택 관련 소득공제

#### 1-6-1. 주택청약종합저축 소득공제

> 근거: 조세특례제한법 제87조 제2항

| 항목 | 내용 |
|---|---|
| 공제대상 | 무주택 세대주인 근로자 |
| 총급여 조건 | 7,000만원 이하 |
| 공제율 | 납입액의 40% |
| 납입한도 | 연 300만원 (월 25만원) |
| 공제한도 | 연 120만원 (300만원 x 40%) |
| 가입상품 | 주택청약종합저축, 청년우대형 주택청약종합저축 |

- 무주택 확인: 배우자 포함 세대 전원 무주택
- 과세기간 중 주택 취득 시: 해당 연도 납입분 공제 불가 (해지 간주)
- 청년우대형: 만 19~34세, 연소득 3,600만원 이하, 무주택 세대주

#### 1-6-2. 주택담보대출 이자상환액 공제

> 근거: 소득세법 제52조 제5항

**적용 조건:**
- 무주택 또는 1주택 세대주
- 취득 당시 기준시가 6억원 이하 (2025년 귀속)
- 상환기간 15년 이상

| 상환기간 및 방식 | 공제한도 |
|---|---|
| 15년 이상, 고정금리 AND 비거치식 | 2,000만원 |
| 15년 이상, 고정금리 OR 비거치식 | 1,800만원 |
| 15년 이상, 기타 (변동금리 + 거치식) | 800만원 |
| 10년 이상 15년 미만, 고정금리 AND 비거치식 | 600만원 |
| 10년 이상 15년 미만, 고정금리 OR 비거치식 | 300만원 |

- **고정금리:** 대출 만기까지 이자율 변동 없는 대출
- **비거치식:** 거치기간 없이 최초 상환일부터 원리금 분할상환
- 2채 이상 보유 시: 공제 불가
- 차입금의 용도: 주택 구입(취득)에 직접 사용

#### 1-6-3. 전세자금 대출 원리금 상환액 공제 (주택임차차입금)

> 근거: 소득세법 제52조 제4항

| 구분 | 내용 |
|---|---|
| 대상 | 무주택 세대주 (또는 세대주가 주택관련 공제를 받지 않는 경우 세대원) |
| 국민주택규모 이하 | 전용면적 85㎡ 이하 |
| 공제대상 | 원리금 상환액의 40% |
| 한도 | 연 400만원 |

**금융기관 차입:**
- 대출기관으로부터 직접 임대인에게 지급된 전세보증금 대출
- 원리금 상환액의 40%, 연 400만원 한도

**개인(대부업 제외) 차입:**
- 이자 상환액, 연 400만원 한도
- 임대차계약서상 확정일자 필수

- 주택담보대출 이자 + 전세자금 원리금 + 주택청약 합산하여 총 공제한도 적용 없음 (각각 별도 한도)

### 1-7. 신용카드 등 사용금액 소득공제

> 근거: 조세특례제한법 제126조의2

#### 기본 구조

**최저사용금액:** 총급여의 25%를 초과하는 금액부터 공제 시작

| 결제수단 | 공제율 |
|---|---|
| 신용카드 | 15% |
| 체크카드 / 현금영수증 | 30% |
| 도서·공연·박물관·미술관 (총급여 7천만원 이하) | 30% |
| 전통시장 | 40% |
| 대중교통 | 80% |

#### 공제한도

| 총급여 구간 | 기본한도 |
|---|---|
| 7,000만원 이하 | 300만원 |
| 7,000만원 초과 ~ 1.2억원 이하 | 250만원 |
| 1.2억원 초과 | 200만원 |

#### 추가한도 (기본한도와 별도)

| 항목 | 추가한도 |
|---|---|
| 전통시장 | 각각 합산 300만원 |
| 대중교통 | 각각 합산 300만원 |
| 도서·공연·박물관·미술관 (총급여 7천만원 이하) | 각각 합산 300만원 |

- 2025년 귀속: 전통시장 + 대중교통 + 도서공연 합산 추가한도 300만원

#### 공제 제외 대상
- 자동차 구입비 (중고차는 10% 공제)
- 국세·지방세, 전기/수도/가스/전화/인터넷 요금
- 아파트 관리비
- 상품권 구입비
- 보험료
- 학원비 (취학전 아동 학원비는 교육비 세액공제)
- 해외 사용분
- 현금서비스

#### 공제 순서
1. 최저사용금액(25%) 초과분 중 **신용카드** 사용분부터 차감
2. 나머지 체크카드/현금영수증/전통시장/대중교통 순서

| 항목 | 내용 |
|---|---|

## 2. 세액공제 전체 항목

### 2-1. 근로소득 세액공제 (자동 적용)

> 근거: 소득세법 제59조

| 산출세액 구간 | 세액공제액 |
|---|---|
| 130만원 이하 | 산출세액의 55% |
| 130만원 초과 | 71.5만원 + 130만원 초과분의 30% |

| 총급여 구간 | 공제한도 |
|---|---|
| 3,300만원 이하 | 74만원 |
| 3,300만원 초과 ~ 7,000만원 이하 | 74만원 - [(총급여 - 3,300만원) x 0.8%], 최소 66만원 |
| 7,000만원 초과 | 66만원 - [(총급여 - 7,000만원) x 50%], 최소 50만원 |

### 2-2. 자녀세액공제

> 근거: 소득세법 제59조의2

#### 기본 자녀세액공제

| 자녀 수 | 공제액 |
|---|---|
| 1명 | 15만원 |
| 2명 | 35만원 (15만 + 20만) |
| 3명 이상 | 35만원 + 3명째부터 1인당 30만원 추가 |

- 대상: 기본공제 대상 자녀 및 손자녀 (만 8세 이상, 만 20세 이하)

- 만 7세 미만은 아동수당 수급 대상이므로 자녀세액공제 제외
- 만 8세 이상 ~ 만 20세 이하

#### 출생·입양 세액공제

| 구분 | 공제액 |
|---|---|
| 첫째 | 30만원 |
| 둘째 | 50만원 |
| 셋째 이상 | 70만원 |

- 해당 과세기간에 출생 또는 입양 신고한 경우

### 2-3. 연금계좌 세액공제

> 근거: 소득세법 제59조의3

#### 납입한도

| 구분 | 한도 |
|---|---|
| 연금저축 | 연 600만원 |
| 연금저축 + IRP 합산 | 연 900만원 |

#### 공제율

| 총급여 구간 | 공제율 |
|---|---|
| 5,500만원 이하 (종합소득금액 4,500만원 이하) | 15% |
| 5,500만원 초과 | 12% |

#### 최대 공제액

| 총급여 | 연금저축만 | 연금저축 + IRP |
|---|---|---|
| 5,500만원 이하 | 90만원 (600만 x 15%) | 135만원 (900만 x 15%) |
| 5,500만원 초과 | 72만원 (600만 x 12%) | 108만원 (900만 x 12%) |

- 50세 이상(총급여 1.2억 이하): 연금저축 납입한도 600만원은 동일 (2023년 개정으로 전 연령 600만원 통일)
- 퇴직연금(IRP)만 가입 시에도 900만원 한도 적용 가능

### 2-5. 의료비 세액공제

> 근거: 소득세법 제59조의4 제2항

#### 기본 구조
- 의료비 총액이 **총급여의 3%** 초과하는 금액에 대해 공제
- 대상: 본인 + 기본공제 대상자 (나이·소득 제한 없이 의료비는 적용 가능)

#### 세부 공제율 및 한도

| 구분 | 공제율 | 한도 |
|---|---|---|
| 본인 | 15% | 한도 없음 |
| 만 65세 이상 | 15% | 한도 없음 |
| 장애인 | 15% | 한도 없음 |
| 난임시술비 | 30% | 한도 없음 |
| 미숙아·선천성이상아 | 20% | 한도 없음 |
| 일반 (위 외 부양가족) | 15% | 연 700만원 |

#### 3% 초과분 적용 순서
1. 본인/65세이상/장애인/난임/미숙아 → 한도없음 항목 우선 차감
2. 나머지 → 일반 의료비 (700만원 한도)

#### 공제 포함 항목
- 진찰, 치료, 수술, 입원, 약제비
- 한약 (한의원 처방)
- 안경·콘택트렌즈 (1인당 50만원 한도)
- 보청기, 장애인보조기구
- 산후조리원 (총급여 7,000만원 이하, 출산 1회당 200만원 한도)
- 라식/라섹 수술비
- 치과 치료비 (임플란트, 교정 등)
- 보철, 틀니

#### 공제 제외 항목
- 미용·성형수술비
- 건강검진비: 본인 부담분은 **공제 가능** (건강보험공단 전액 지원 검진은 해당 없음)
- 간병비
- 외국 의료기관 지출분
- **실손보험 수령액은 반드시 차감** (2019년 귀속부터)

## 3. 비과세 소득 전체

> 근거: 소득세법 제12조, 소득세법 시행령 제12조~제18조

### 3-1. 식대 (비과세 식사대)

| 항목 | 내용 |
|---|---|
| 비과세 한도 | 월 20만원 |
| 조건 | 사내급식 등을 제공받지 않는 근로자에게 지급하는 식사대 |
| 근거 | 소득세법 시행령 제17조의2 |

- 2023년부터 월 10만원 → 20만원으로 상향
- 식대를 받으면서 사내식당도 이용 시: 비과세 불가

### 3-2. 자가운전보조금

| 항목 | 내용 |
|---|---|
| 비과세 한도 | 월 20만원 |
| 조건 | 본인 소유 차량을 업무에 직접 사용하고 출장여비 등을 받는 경우 |
| 근거 | 소득세법 시행령 제12조 제3호 |

- 시내출장에 한정 (일반 출퇴근 아님, 업무용 차량 운행)
- 차량유지비, 유류대 등 포함

### 3-3. 육아수당 (출산·보육수당)

| 항목 | 내용 |
|---|---|
| 비과세 한도 | 월 20만원 |
| 조건 | 만 6세 이하 자녀가 있는 근로자 |
| 근거 | 소득세법 제12조 제3호 러목 |

- 부부 각각 적용 가능 (각자 월 20만원)
- 2025년 귀속: 월 20만원 유지

### 3-4. 연구보조비·연구활동비

| 항목 | 내용 |
|---|---|
| 비과세 한도 | 월 20만원 |
| 조건 | 유아교육법/초중등교육법/고등교육법에 따른 학교 및 이에 준하는 학교 교원, 연구기관 종사 연구활동종사자 |
| 근거 | 소득세법 시행령 제12조 제11호 |

### 3-5. 야간근로수당 등 (생산직 근로자)

| 항목 | 내용 |
|---|---|
| 비과세 한도 | 연 240만원 |
| 조건 | 생산직 등 근로자, 월정액급여 210만원 이하, 직전 과세기간 총급여 3,000만원 이하 |

- 일용근로자의 경우: 전액 비과세 (별도)

### 3-9. 출산지원금 비과세 (2024년 신설)

## 4. 자주 묻는 질문 (FAQ)

### Q1. 맞벌이 부부 — 의료비 몰아주기 가능한가?

**가능하다.** 의료비 세액공제는 소득·나이 요건 없이 "생계를 같이하는 부양가족"이면 공제 가능하다.
- 배우자 소득이 있어도 한쪽에서 배우자의 의료비를 공제받을 수 있음
- 단, **의료비만 가능**. 카드결제 건은 카드 명의자 기준.
- 전략: 소득이 낮은 쪽이 의료비를 몰아서 공제하면 3% 기준금액이 낮아져 유리

### Q2. 맞벌이 부부 — 부양가족 등록

- 부양가족 기본공제: 배우자 소득이 연 100만원(총급여 500만원) 초과 시, 서로 기본공제 불가
- 자녀: 한쪽만 기본공제 등록 (중복 불가)
- 부모님: 한쪽만 등록 (형제자매 중 한 명만)
- **원칙:** 기본공제 등록된 쪽에서 해당 부양가족의 보험료·의료비·교육비·카드 공제도 받음 (의료비 제외 — 의료비는 몰아주기 가능)

### Q3. 맞벌이 부부 — 자녀 공제 중복

- 자녀 기본공제: 부부 중 한쪽만 등록 (동일 자녀를 양쪽에서 공제 불가)
- 자녀세액공제: 기본공제 등록한 쪽에서만 가능
- 자녀 교육비/의료비: 기본공제 등록자 쪽에서 공제 (의료비는 몰아주기 가능)

### Q4. 태아보험 — 공제 가능한가?

**불가.** 태아는 기본공제 대상자가 아님 (출생 전).
- 출생 후 보험료를 납부한 분부터 공제 가능
- 임신 중 납부한 태아보험료는 공제 불가

### Q5. 실손보험 — 공제

- 실손보험료 납입액: **보장성보험료 세액공제 가능** (연 100만원 한도 내)
- 실손보험 수령액: **의료비에서 차감** 필수
- 즉, 실손보험으로 받은 금액은 의료비 공제 대상에서 빠짐

### Q6. 안경·콘택트렌즈 — 공제 가능한가?

**가능하다.**
- 의료비 세액공제 대상
- 1인당 연 50만원 한도
- 시력보정용에 한함 (선글라스 등 제외)
- 안경점에서 "시력보정용 안경 구입 영수증" 발급 필요

### Q7. 라식·라섹 — 공제 가능한가?

**가능하다.**
- 의료비 세액공제 대상
- 시력교정 목적의 수술이므로 의료비에 해당
- 50만원 한도 아님 (안경과 별도, 일반 의료비로 분류)

### Q8. 산후조리원 — 공제 가능한가?

**가능하다.**
- 총급여 7,000만원 이하인 근로자
- 출산 1회당 200만원 한도
- 산후조리원 이용 영수증 필요

### Q9. 정치자금 기부 — 실질 혜택은?

- 10만원 기부 시: 전액 환급 (100/110 세액공제, 약 10만원 돌려받음)
- 가장 확실한 절세 방법 중 하나
- 정당 당원가입 or 후원회 후원금으로 가능
- 선관위 기부금 영수증 자동 발급 (간소화 연동)

### Q10. 공동명의 주택과 세대주 조건

- 주택청약 소득공제: 무주택 **세대주**만 가능
- 공동명의 주택 보유 시: 유주택자로 간주 → 청약공제 불가
- 주담대 이자공제: 세대주 + 1주택 + 기준시가 6억 이하
- 공동명의라도 1주택 요건 충족 시 공제 가능 (세대주 명의 대출)

### Q11. 전세 vs 월세 — 어떤 공제?

| 구분 | 전세 | 월세 |
|---|---|---|
| 공제 유형 | 소득공제 (전세자금대출 원리금 상환) | 세액공제 |
| 대상 | 전세자금 대출받은 경우 | 월세 납부자 |
| 조건 | 무주택 세대주 | 무주택 세대주, 총급여 8,000만원 이하 |
| 공제액 | 원리금 상환액의 40%, 최대 400만원 | 월세의 15~17%, 최대 1,000만원 |

- 전세: 대출 없이 전세금 자체로는 공제 없음
- 월세: 세액공제 또는 현금영수증 소득공제 중 택 1 (중복 불가)

### Q12. 중도입사·퇴사 시 연말정산

- **중도입사:** 입사 전 소득은 전 직장에서 원천징수영수증 받아서 현 직장에 제출 → 합산 정산
- **중도퇴사:** 퇴사 시 회사에서 중도퇴사 연말정산 진행. 이후 다른 소득 있으면 5월 종합소득세 신고.
- 퇴사 시점까지의 소득만 정산
- 부족한 공제는 5월 종합소득세에서 추가 신청 가능

### Q13. 이직 시 연말정산

- 현 직장에서 전 직장 원천징수영수증 합산하여 정산
- 전 직장 원천징수영수증: 전 회사에 요청 or 홈택스에서 발급
- 연중 2개 이상 직장 근무 시: 마지막 직장에서 합산 정산
- 만약 합산하지 않았다면: **5월 종합소득세 신고** 시 합산

### Q14. 프리랜서 겸업 시

- 근로소득 + 사업소득(3.3% 원천징수): 5월 종합소득세 신고 필수
- 연말정산은 근로소득분만 진행
- 사업소득분은 5월에 합산 신고
- 기납부세액(3.3% 원천징수분)은 종소세에서 차감

### Q15. 부모님 부양가족 등록 조건

- 만 60세 이상 + 연소득 100만원 이하
- 형제자매 중 **1명만** 등록 가능 (중복 불가)
- 주민등록상 동거 또는 별거 시 실질 부양 입증
- 시부모·장인장모도 가능 (직계존속)
- 배우자의 부모도 가능 (배우자 통해 직계존속)

### Q16. 장애인 증명 범위

- 「장애인복지법」에 따른 등록 장애인
- 「국가유공자법」에 따른 상이자 (1~7급)
- **항시 치료를 요하는 중증환자:** 의료기관에서 "장애인증명서" 발급
  - 암환자, 중풍, 치매, 난치성질환 등
  - 세법상 장애인 = 복지법 장애인보다 넓은 범위

### Q17. 학원비 — 언제 공제 가능?

- **취학전 아동만** 가능 (미취학 아동 학원비: 교육비 세액공제)
- 초등학생 이상: 학원비 공제 불가
- 체육시설(태권도, 수영 등) 포함 (취학전 아동)
- 방과후수업료는 학원비가 아님 (초중고 교육비로 공제 가능)

### Q18. 해외유학 — 교육비 공제 가능?

**조건부 가능.**
- 국외 교육기관에서 교육받는 경우
- 조건: 「국외유학에 관한 규정」에 적합한 유학
  - 자비유학 요건: 중학교 졸업 이상 학력
  - 유학 자격 불문: 국외 근무자의 자녀
- 한도: 국내와 동일 (취학전 300만원, 초중고 300만원, 대학 900만원)

### Q19. 카드 해외사용분 — 공제 가능?

**불가.**
- 신용카드 등 소득공제: **국내 사용분만** 해당
- 해외 직접 사용, 해외 직구(해외결제) 모두 불가
- 국내 면세점 사용분: 공제 가능

### Q20. 부양가족 카드 사용분 합산?

**기본공제 대상자의 카드 사용분은 합산 가능.**
- 조건: 기본공제 대상으로 등록된 부양가족
- 배우자 소득 있어서 기본공제 못 받으면 → 배우자 카드 합산 불가
- 자녀(기본공제 등록): 자녀 명의 카드 사용분 합산 가능
- 부모님(기본공제 등록): 부모님 명의 카드 사용분 합산 가능

### Q21. 결혼세액공제 — 2025년에 혼인신고하면?

**가능하다.** 2024~2026년 중 혼인신고한 부부에게 각각 50만원 세액공제.
- 혼인신고일 기준 (결혼식 날짜 아님)
- 초혼·재혼 불문, 생애 1회 한정
- 부부 각각 본인의 연말정산에서 50만원씩 적용

### Q22. 회사에서 출산축하금을 받았는데 비과세인가?

**2024년 이후 지급분부터 전액 비과세.** (소득세법 제12조 제3호)
- 출생일 이후 2년 이내에 사용자로부터 받는 급여
- 금액 한도 없음 (전액 비과세)
- 최대 2회까지 비과세 가능
- 기존 출산·보육수당(월 20만원)과 별도

## 7. UX 관점 — 사용자 선택 과다 문제 해결 방안

### 7-1. 문제 정의

연말정산은 항목이 50개 이상이고, 각 항목마다 조건이 복잡하다. 사용자가 모든 항목을 직접 확인하면 이탈률이 높아진다.

### 7-2. 단계별 안내 방식 (질문-답변형 위저드)

```
Step 1: 기본정보 입력
  - 총급여 (급여명세서 기준)
  - 결혼 여부 (미혼/기혼)
  - 자녀 수 + 나이
  - 주택 보유 여부 (무주택/1주택/다주택)
  - 세대주 여부

Step 2: 자동 판별
  → 해당 항목만 필터링하여 표시
  → 미해당 항목은 자동 숨김

Step 3: 항목별 상세 입력
  → 해당되는 항목만 순차 안내
  → 각 항목에 "이게 뭔가요?" 툴팁 제공

Step 4: 결과 요약
  → 예상 환급액 / 추가납부액
  → 놓친 항목 체크리스트
```

### 7-3. 프로필 기반 자동 추천

| 프로필 | 핵심 공제항목 |
|---|---|
| 미혼, 무주택, 사회초년생 | 주택청약, 카드공제, 연금저축, 월세, 중소기업감면 |
| 기혼, 외벌이, 자녀 있음 | 인적공제, 자녀세액공제, 교육비, 의료비, 보험료 |
| 기혼, 맞벌이, 자녀 있음 | 의료비 몰아주기, 자녀 배분 최적화, 연금저축 |
| 기혼, 주택 보유 | 주담대 이자공제, 보험료, 기부금 |
| 고소득 (1.2억 초과) | 연금저축+IRP, 기부금, 정치자금, 카드한도 축소 주의 |
| 부모님 부양 | 인적공제(경로우대), 의료비, 보험료 |

### 7-4. 필수항목 / 선택항목 구분

### 7-5. 해당 안 되는 항목 자동 숨김 로직

```
IF 미혼 → 배우자공제, 부녀자공제 숨김
IF 자녀 없음 → 자녀세액공제, 교육비(자녀), 출산입양공제 숨김
IF 유주택 → 주택청약, 월세, 전세자금대출 숨김
IF 다주택 → 주택담보대출이자 숨김
IF 총급여 > 7,000만원 → 주택청약, 산후조리원, 도서공연공제 숨김
IF 총급여 > 8,000만원 → 월세세액공제 숨김
IF 취학전 자녀 없음 → 학원비 교육비 숨김
IF 65세 이상 부양가족 없음 → 경로우대공제 숨김
IF 장애인 부양가족 없음 → 장애인공제 숨김
```

### 7-6. 절세 효과 우선순위 (ROI 기준 추천)

사용자에게 "가성비 좋은 절세" 순서로 안내:

1. **정치자금 기부 10만원** — 전액 환급 (ROI 100%)
2. **연금저축 + IRP 납입** — 최대 135만원 세액공제
3. **중소기업 취업자 감면** — 소득세 90% 감면 (해당자)
4. **월세 세액공제** — 최대 170만원
5. **인적공제 최적화** — 1인당 150만원 소득공제
6. **의료비 몰아주기** — 맞벌이 전략
7. **카드 사용 최적화** — 체크카드/현금영수증 비중 높이기
8. **기부금** — 고액 기부 시 절세 효과 큼
